== Description ==

Plagin «Online.ua From WP Posting» you can influence the automatic publication of records from Your Wordpress of bloga on the on-line-wall.	  For tuning and confirmation of automatic publication of records you must get plagin «Online.ua From WP Posting» and set him in the bloge of Wordpress. After setting in tuning of plagina it is necessary to specify the code of activating, placed below. If a code will be changed or an automatic publication will absent executed will not be.

== Installation ==

Upload the Online.ua From WP Posting plugin to your blog, Activate it, then enter your hash code.
